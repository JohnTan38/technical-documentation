Introduction

A simple tool to comapre 2 csv files. The end result is saved in a new csv file.

Installation

The user guide assumes basic level knowledge of app execution. Prior experience with other frameworks helps, but is not required.

The easiest way to start is to download the executable (.exe) app:

Usage

Download the .exe file and save in Downloads folder. Save a copies of 'our_data.xlsx', 'cust_data.xlsx', and 'cust_diff.csv' in Downloads folder.

Folder structure

C:/Users/your.name/Downloads/our_data.xlsx C:/Users/your.name/Downloads/cust_data.xlsx C:/Users/your.name/Downloads/cust_diff.xlsx C:/Users/your.name/Downloads/app

Now we can run the app and compare.

Run the .exe file

Double-click the .exe file to run the app.

  1. Python code executes in background. Result is saved to cust_diff.xlsx
    • Chrome updates automatically in the background. Current versions for Windows: Chrome 131+
    • Data input from user populated csv files.
    • When process is completed, differences saved in cust_diff.xlsx
  2. Process can be scheduled to run on a regular basis. Alternatively, the .exe file can be run manually on-demand.
